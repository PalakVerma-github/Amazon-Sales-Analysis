
# 📊 Amazon Sales Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing Amazon sales data using Python. It includes data visualization, statistical analysis, and a machine learning model to predict sales amount.

---

## 🎯 Objectives

* Analyze sales trends across different time periods
* Compare performance across product categories and regions
* Identify factors affecting sales
* Build a prediction model using Linear Regression
* Perform hypothesis testing for business insights

---

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* SciPy

---

## 📊 Features

* Data analysis and insights
* Bright and colorful visualizations
* Correlation analysis
* Sales prediction using Linear Regression
* Hypothesis testing (t-test, chi-square, Shapiro test)

---

## 📁 Dataset

The dataset is provided in Excel format and included in this repository.

---

## ▶️ How to Run

1. Clone or download this repository
2. Install required libraries:
   pip install -r requirements.txt
3. Run the Python file:
   python main.py
